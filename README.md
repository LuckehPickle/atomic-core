# Atomic Core
The core plugin for the Atomic Hive Minecraft server.

## Permission nodes
These are all the permission nodes within Atomic Core. Alphabetical order by commands.

| Command        | Permission              |
| :------------- | :---------------------- |
| all            | atomic-core.*           |
| /ping          | atomic-core.ping        |
| /ping [player] | atomic-core.ping.others |
